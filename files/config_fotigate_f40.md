# Come configurare Fotigate F40

Il Fortigate F40 e' un firewall prodotto da Fortinet, utilizzato per proteggere le reti e gestire il traffico di rete.

## Passaggi per la configurazione

## Spigazione voci del menu

### Network

#### DNS

Per configurare il DNS, andare su Network > DNS e impostare i server DNS desiderati.
Se si vuole impostare i propri server DNS, selezionare "Specify" e inserire gli indirizzi IP dei server DNS preferiti.

### Policy & Objects

#### Firewall Policy

##### Configurazione di una regola di firewall

Per configurare la regola di firewall da "LAN_to_WAN_OUT", seguire questi passaggi:

1. Abilitare il NAT
2. Abilitare AntiVirus (default)
3. Abilitare Web Filter (default)
4. Abilitare DNS Filter (default)
