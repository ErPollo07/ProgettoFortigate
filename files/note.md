# Note

## Statefull firewall

Il fortigate è un firewall statefull, questo significa che è in grado di tenere traccia dello stato delle connessioni. Quando un pacchetto entra nel firewall, il fortigate analizza le informazioni contenute nel pacchetto e le confronta con le regole di sicurezza configurate. Il firewall tiene traccia delle connessioni e se c'è una firewall policy che consente il traffico allora non serve configurare una regola di ritorno.
